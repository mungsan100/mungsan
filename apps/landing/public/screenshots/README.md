# 앱 스크린샷 넣는 곳

이 폴더에 스크린샷 이미지 파일(png / jpg)을 넣은 뒤,
`apps/landing/lib/landing-config.ts` 의 `SCREENSHOTS` 에서 해당 항목의
`src` 를 `'/screenshots/파일이름.png'` 로 바꾸면 회색 placeholder 대신
실제 이미지가 표시됩니다.

| 설정 키       | 들어갈 화면                          | 섹션                |
| ------------- | ------------------------------------ | ------------------- |
| `howItWorks`  | 파트너 추천 / 협업 마켓플레이스 화면 | 5. 작동 원리        |
| `lounge`      | 임원 전용 익명 라운지 앱 화면        | 6. 임원 전용 라운지 |
| `marketplace` | 협업 마켓플레이스 앱 화면            | 7. 협업 마켓플레이스 |
| `sherpa`      | 공유 대시보드 / 공동 수행 관리 화면  | 8. My 셰르파        |
| `dataGraphic` | 데이터 기반 파트너 추천 구조 그래픽  | 9. 데이터 기반 분석 |
